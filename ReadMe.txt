========================================================================
    КОНСОЛЬНОЕ ПРИЛОЖЕНИЕ. Обзор проекта Mathematical Parser
========================================================================

Данное приложение представляет из себя парсер математических выражений, содержащих бинарные операторы +, -, *, /.
Унарные операторы не воспринимаются приложением корректно.

Содержит классы, помогающие преобразовать выражение из инфиксной формы записи в польскую нотацию.
Вычислитель принимает на вход "выражение" в обратной польской нотации и находит значение выражения.

Проект компилируется в среде разработки Visual Studio 2015 Community Edition. Для компиляции необходимо открыть файл, имеющий расширение .sln.

В зависимости от опций компиляции исполняемый файл приложения располагается в субдиректориях /Debug/ или /Release/.

Допустимые форматы 
-a+b
a+b
b+(c/d)
(a*(c+(b/d)))
a++b
b+-c
в случае встреченых подряд операторов + и - они преобразуются в единственный бинарный оператор.

В случае, когда выражение содержит недопустимые символы будет выведено сообщение Tokenizing Error. В случае ошибки вычисления "Something gone wrong"

Выражение вводится в инфиксной форме. Программа выводит выражение в польской нотации и результат.

Для выхода из приложения введите "exit".

Пробелы не валидны.
